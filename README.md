## 📌 Library Management System (Kotlin - Console Based)

🚀 [Click here to run the application live in your browser!](https://replit.com/@jeyasuriya01/LibraryManagementSystem18020)

A beginner-friendly, console-based Library Management System written in Kotlin. This project demonstrates core object-oriented programming concepts using dynamic data structures like MutableList. It allows users to efficiently manage books, students, book issuing/returning, and fine calculation through a simple menu-driven interface.

---

## ✨ Features

* **Dashboard:** View total books, students, issued books, and available books.
* **Book Management:** Add, view, search, update, and delete books.
* **Student Management:** Add, view, search, update, and delete student records.
* **Issue Book:** Issue books to students with due days.
* **Return Book:** Return books and automatically calculate late fine.
* **View Issued Books:** Display all currently issued books with student details.
* **Fine Calculation:** Calculate fine based on late days.
* **Pending Fines:** Display basic information about fine system.
* **Menu-Driven Interface:** Simple and user-friendly console navigation.

## ▶️ How to Run Locally

1. **Save the Kotlin file** (e.g., `LibrarySystem.kt`)

2. **Open Command Prompt / Terminal**

3. **Navigate to the folder**

```bash
cd path/to/your/folder
```

4. **Compile the Kotlin file**

```bash
kotlinc LibrarySystem.kt
```

5. **Run the program**

```bash
kotlin LibrarySystemKt
```
